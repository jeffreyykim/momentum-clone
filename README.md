# Momentum Clone

A browser extension that mimics the Momentum dashboard, featuring a clean interface with time display, personalized greetings, and inspirational quotes.

## Getting Started

Install the dependencies and run the project:
```
npm install
npm start
```

## Features

- Current time display (updates every second)
- Random nature background images from Unsplash
- Dogecoin price tracker (current price, 24h high/low)
- Local weather display based on geolocation
- Clean, minimalist design
- Chrome extension that replaces new tab page
